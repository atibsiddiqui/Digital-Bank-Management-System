# Secure Bank Management System

A Java-based console application that automates basic banking operations.

## Features
- Account creation
- Deposit and withdrawal
- Balance inquiry
- Account management

## Technologies Used
- Java
- Object-Oriented Programming

## How to Run
1. Compile the program using `javac BankManagementSystem.java`
2. Run using `java BankManagementSystem`
